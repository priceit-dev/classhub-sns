# SSOT 참조 룰

## 기본 원칙

모든 콘텐츠는 [classhub-wiki](https://github.com/priceit-dev/classhub-wiki)를 **단일 진실 공급원(SSOT)**으로 참조하여 작성합니다.

Wiki에 없는 내용은 콘텐츠에 넣지 않습니다. Wiki와 충돌하는 내용은 발행하지 않습니다.

## 콘텐츠 작성 시 참조 섹션

| 콘텐츠 유형 | Wiki 참조 섹션 | 핵심 확인 사항 |
|-------------|-------------|---------------|
| 숏폼/릴스 | 5.4 채널별 콘텐츠 가이드 | 톤: 임팩트, 후킹 |
| 카페 글 | 5.5 카페 게시판 구조 | 게시판 카테고리 맞춤 |
| 오픈채팅방 | 5.3 채널 역할 (MOFU) | 톤: 친근, 대화체 |
| 카드뉴스 | 2.1 고객 세그먼트 | 타깃 페르소나 특정 |
| CTA 콘텐츠 | 5.1 마케팅 퍼널 | 퍼널 단계 맞춤 |
| 성공 사례 | 5.7 성장 루프 | 루프 연결고리 = 성공 사례 |

## 타깃 페르소나 (Wiki 2.1절)

콘텐츠 작성 전 반드시 타깃을 특정합니다:

| 세그먼트 | Pain Point | 콘텐츠 방향 |
|-----------|-----------|----------|
| 직장인 N잡러 | 시간 부족, 부수입 필요 | "퇴근 후 2시간으로" |
| 전업주부 | 경제적 독립, 시간 활용 | "집에서 할 수 있는" |
| 대학생 | 경험+수익 동시 | "실전 경험으로 취업까지" |

## 금지사항 (Wiki 기반)

- 핵심 가치 7개와 충돌하는 메시지 금지 (Wiki 1.4)
- "러너"를 "수강생"으로 잘못 표기 금지 (Wiki 7장 용어사전)
- 무료강의 신청자 ≠ 러너 혼동 금지
- 코치 = 계약 파트너 ("직원"이라고 하지 않음)
- 기밀 정보 (가격, 정산, 전환율 구체 수치) 포함 금지

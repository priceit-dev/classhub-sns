# 네이버 카페 글 작성 가이드

## SEO 원칙
- 제목에 핵심 키워드 포함 (위탁판매, 식품셀러, 공급가 등)
- 본문 1,000자 이상 (네이버 검색 노출 기준)
- 이미지 3장 이상
- 관련 글 내부 링크

## 글 구조
```
[제목] 키워드 포함, 궁금증 유발

[인트로] 2~3줄로 핵심 요약

[본문]
- 소제목 3~4개로 구분
- 각 소제목 아래 3~5줄
- 숫자/데이터 강조
- 이미지 삽입

[CTA] 수강 신청 링크 또는 오픈채팅방 링크
```

## 시리즈 예시
1. "컨퍼런스 발표 핵심 정리 ① — 식품 위탁판매 시장 현황"
2. "컨퍼런스 발표 핵심 정리 ② — 공급가 비교의 중요성"
3. "컨퍼런스 발표 핵심 정리 ③ — 성공 셀러의 3가지 습관"
4. "컨퍼런스 발표 핵심 정리 ④ — 클래스허브로 시작하기"

## 금지사항
- 과장 광고 표현 ("무조건", "보장")
- 경쟁사 비방
- 개인정보 노출

# 숏폼 자동화 파이프라인 기획서

> 세션53 (2026-03-17) 확정
> 상태: 1단계 구현 시작

## 목적

영상 콘텐츠를 100% 자동으로 숏폼으로 변환.
편집 인력 없이도 대량 생산 가능한 파이프라인.

## 배경

- 클래스허브 SNS 마케팅 본격화 (9개 채널)
- 컨퍼런스 영상 등 소스 영상 지속 생산 예정
- 편집 인력 부재 → 100% 자동화 필수
- 향후 영상 자체도 AI 생성 예정

## 1단계: `/shortform` 스킬 — 기존 영상 → 숏폼

### 파이프라인

```
입력: 영상 파일 경로

Step 1. Whisper API (large-v3)
  - word-level timestamp 추출
  - 비용: $0.006/분

Step 2. Claude Opus
  - 자막 한국어 교정 (오인식 수정)
  - 숏폼 구간 선정 (30~60초 단위)
    - 감정 곡선 분석
    - 후킹 포인트 판별
    - 스토리 아크 완결성
  - 채널별 카피 생성
    - 인스타 릴스: 제목 + 설명 + 해시태그 30개
    - 유튜브 숏츠: 제목 + 설명
    - 틱톡: 제목 + 해시태그
  - 썸네일 카피 3안
  - 비용: ~$0.80

Step 3. Creatomate API
  - 9:16 세로 리프레임
  - 애니메이션 자막 (단어별 하이라이트)
  - 브랜드 인트로/아웃트로 (템플릿)
  - 페이드 트랜지션
  - 비용: $0.06~0.12/렌더

출력: ~/Downloads/shortform-{날짜}/
  ├── clips/           # 완성된 숏폼 영상
  ├── subtitles/       # 교정된 SRT 파일
  ├── content.md       # 채널별 카피
  ├── thumbnails.md    # 썸네일 카피안
  └── timeline.md      # 구간 조정용
```

### 총 비용
- ~$1.00/숏폼 (5분 영상 기준)
- 사람 개입: 0% (불만족 시 파라미터 조정 후 재렌더)

### 재렌더 흐름
```
사람이 timeline.md에서 시작/끝 시간 수정
→ /shortform reclip
→ 수정된 구간으로 재렌더
```

## 2단계: `/ai-video` — AI 영상 자체 제작

### 파이프라인

```
입력: 주제 또는 스크립트

Step 1. Claude Opus
  - 숏폼 대본 작성
  - 장면 구성 (컷 단위)

Step 2. 영상 생성
  - HeyGen API: AI 아바타 발화 (한국어, ~$30/월)
  - Runway Gen-3: 감성 컷/B롤 ($0.05/초)
  - Kling: 고품질 AI 영상 (~$10/월)

Step 3. Creatomate API
  - 자막 + 편집 + 렌더링

출력: 업로드 가능한 완성 영상
```

### 총 비용: ~$2~3/영상

## 기술 선택 근거

### Creatomate 선택 이유
- REST API → 스킬에서 프로그래밍 호출 가능
- JSON 기반 템플릿 → 한 번 만들면 수백 개 동일 퀄리티
- 자막 애니메이션 퀄리티가 CapCut급
- 렌더당 $0.06~0.12 저렴
- 브랜드 템플릿으로 통일된 아이덴티티

### 비교 검토한 서비스

**기존 영상 → 숏폼:**
| 서비스 | 비용 | API | 판단 |
|--------|------|-----|------|
| Opus Clip | ~$20/월 | 있음 | 편하지만 커스터마이징 한계 |
| Vizard.ai | ~$30/월 | 있음 | 한국어 우수, API 자유도 중간 |
| Creatomate | $0.06/렌더 | REST | ✅ 자유도 최고, 템플릿 재사용 |

**AI 영상 제작:**
| 서비스 | 비용 | 용도 |
|--------|------|------|
| HeyGen | ~$30/월 | AI 아바타 (한국어 발화) |
| Synthesia | ~$30/월 | 기업용 AI 아바타 |
| Runway Gen-3 | $0.05/초 | 텍스트/이미지→영상 |
| Kling | ~$10/월 | 고품질 AI 영상 |

## 구현 우선순위

1. Creatomate 가입 + API 키 발급
2. 클래스허브 브랜드 템플릿 생성
3. `/shortform` 스킬 구현
4. 컨퍼런스 영상 v4로 테스트
5. 결과 확인 후 2단계 진입 여부 결정

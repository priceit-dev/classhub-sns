# 콘텐츠 채널 매트릭스

## 원칙: 1원본 → 전채널 변환

영상 1개, 글 1개를 만들면 → 모든 채널에 형식 변환해서 발행.

## 채널별 역할

| 채널 | 형식 | 특성 | 빈도 | 담당 |
|------|------|------|------|------|
| **인스타 릴스** | 숏폼 30초~1분 | 신규 유입, 바이럴 | 주 3회 | 예진 |
| **유튜브 쇼츠** | 숏폼 30초~1분 | 검색 노출, 장기 유입 | 주 3회 | 예진 |
| **유튜브 본편** | 영상 5~15분 | 깊이, 신뢰 구축 | 월 2회 | 예진+지은 |
| **틱톡** | 숏폼 15초~1분 | MZ 유입, 트렌드 | 주 3회 | 예진 |
| **네이버 카페** | 블로그형 글 | SEO, 검색 유입 | 주 2회 | 지은 |
| **네이버 블로그** | 정보성 글 | SEO, 브랜딩 | 주 1회 | 지은 |
| **오픈채팅방** | 요약+링크 | 직접 전환 (2,600명) | 주 1회 | 송이사 |
| **디스코드** | 커뮤니티 글 | 수강생 유지 | 수시 | 전체 |
| **트위터/X** | 짧은 인사이트 | 업계 네트워킹 | 주 2회 | 지은 |

## 1원본 → 전채널 변환 플로우

### 영상 원본 (15분 컨퍼런스)
```
영상 원본
  ├─ 숏폼 8~10개 (인스타/유튜브쇼츠/틱톡 동시 업로드)
  ├─ 유튜브 본편 1개 (15분 원본 또는 편집본)
  ├─ 카페글 시리즈 4~5개 (네이버 카페 + 블로그)
  ├─ 카드뉴스 3~4세트 (인스타 피드)
  ├─ X 트윗 3~5개 (핵심 인사이트)
  └─ 오픈채팅방 요약 1개
```

### 글 원본 (카페 글 1개)
```
글 원본
  ├─ 네이버 카페 (원문)
  ├─ 네이버 블로그 (약간 수정)
  ├─ 카드뉴스 (핵심 데이터 시각화)
  ├─ X 트윗 (핵심 1줄)
  └─ 디스코드 (토론 유도)
```

## 주간 콘텐츠 최소 발행량

| 채널 | 주간 최소 | 월간 |
|------|---------|------|
| 인스타 릴스 | 3개 | 12개 |
| 유튜브 쇼츠 | 3개 | 12개 |
| 유튜브 본편 | 0.5개 | 2개 |
| 틱톡 | 3개 | 12개 |
| 네이버 카페 | 2개 | 8개 |
| 네이버 블로그 | 1개 | 4개 |
| X | 2개 | 8개 |
| 오픈채팅방 | 1개 | 4개 |
| **총** | **15.5개** | **62개** |

> 숏폼은 인스타/유튜브/틱톡 동시 업로드로 실제 제작량은 주 3개
> 글은 카페/블로그 변환으로 실제 작성량은 주 2개
>
> **실제 제작량: 주 5개 원본 → 15개+ 발행**

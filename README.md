# 클래스허브 SNS 마케팅

식품 위탁판매 교육 플랫폼 **클래스허브**의 SNS 마케팅 관리 레포입니다.

## 팀
| 이름 | 역할 |
|------|------|
| 이상훈 대표 | 방향/승인 |
| 송현규 이사 | 총괄, 성과 리뷰 |
| 예진 | 디자인/영상 (숏폼, 카드뉴스) |
| 김지은 | 기획/작성 (카페 글, 캘린더) |

## 구조
```
specs/          # 기획서
calendar/       # 주간 콘텐츠 캘린더
content/        # 콘텐츠 초안 (카페글, 카드뉴스 등)
  cafe/         # 네이버 카페 글
  reels/        # 릴스/숏폼 스크립트
  cards/        # 카드뉴스
assets/         # 영상, 이미지 원본
guides/         # 제작 가이드
```

## 워크플로우
1. `calendar/`에 주간 일정 확인
2. `content/` 폴더에 초안 작성 (PR)
3. 리뷰 → 승인 → 발행
4. 결과 기록 (조회수, 전환 등)

## 채널
- 인스타 릴스: 주 3회 (신규 유입)
- 네이버 카페: 주 2회 (SEO + 신뢰)
- 오픈채팅방: 주 1회 (직접 전환)
- 디스코드: 수시 (커뮤니티)

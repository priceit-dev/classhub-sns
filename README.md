# 클래스허브 SNS 마케팅

식품 위탁판매 교육 플랫폼 **클래스허브**의 SNS 마케팅 관리 레포.
4월 강의 8개 풀가동 전, 전 채널 SNS 가동이 목표.

## 팀

| 이름 | 역할 | 담당 |
|------|------|------|
| 이상훈 대표 | 방향/승인 | 콘텐츠 방향 결정, 최종 승인 |
| 송현규 이사 | 총괄 | 일정 관리, 오픈채팅방 공유, 성과 리뷰 |
| 예진 | 디자인/영상 | 숏폼 제작, 카드뉴스, 썸네일 |
| 김지은 | 기획/작성 | 콘텐츠 기획, 카페 글, 캘린더 관리 |
| AI 참모 | 자동화 | 글 초안, 자막 추출, 캘린더 생성 |

## 핵심 원칙: 1원본 → 전채널 변환

영상 1개 또는 글 1개를 만들면 → 모든 채널에 형식 변환해서 발행.

```
영상 원본 (15분)
  ├─ 숏폼 8~10개 → 인스타 + 유튜브쇼츠 + 틱톡 동시
  ├─ 유튜브 본편 1개
  ├─ 카페글 시리즈 4~5개 → 네이버 카페 + 블로그
  ├─ 카드뉴스 3~4세트 → 인스타 피드
  ├─ X 트윗 3~5개
  └─ 오픈채팅방 요약 1개

실제 제작량: 주 5개 원본 → 15개+ 발행
```

## 채널 매트릭스

| 채널 | 형식 | 특성 | 빈도 | 담당 |
|------|------|------|------|------|
| 인스타 릴스 | 숏폼 30초~1분 | 신규 유입, 바이럴 | 주 3회 | 예진 |
| 유튜브 쇼츠 | 숏폼 30초~1분 | 검색 노출, 장기 | 주 3회 | 예진 |
| 유튜브 본편 | 영상 5~15분 | 깊이, 신뢰 | 월 2회 | 예진+지은 |
| 틱톡 | 숏폼 15초~1분 | MZ 유입 | 주 3회 | 예진 |
| 네이버 카페 | 블로그형 글 | SEO, 검색 유입 | 주 2회 | 지은 |
| 네이버 블로그 | 정보성 글 | SEO, 브랜딩 | 주 1회 | 지은 |
| 오픈채팅방 | 요약+링크 | 직접 전환 (2,600명) | 주 1회 | 송이사 |
| 디스코드 | 커뮤니티 글 | 수강생 유지 | 수시 | 전체 |
| X (트위터) | 짧은 인사이트 | 업계 네트워킹 | 주 2회 | 지은 |

## 레포 구조

```
specs/              # 기획서
calendar/           # 주간 콘텐츠 캘린더
content/            # 콘텐츠 초안
  ├─ cafe/         # 네이버 카페 글
  ├─ reels/        # 릴스/숏폼 스크립트
  └─ cards/        # 카드뉴스
assets/             # 영상, 이미지 원본
guides/             # 제작 가이드
```

## 워크플로우

1. `calendar/` 에서 이번 주 일정 확인
2. `content/` 에 초안 작성 → PR 생성
3. 리뷰 → 승인 → 발행
4. 발행 후 결과 기록 (조회수, 전환 등)

## Phase 로드맵

**Phase 1 (3/17~3/23)**: 컨퍼런스 영상 활용 시작 — 숏폼 3개 + 카페글 2개 + 오채방 1회

**Phase 2 (3/24~4/6)**: 루틴 구축 — 주간 캘린더 운영, 릴스 주 3회 정기 발행

**Phase 3 (4/7~4/20)**: 4월 강의 오픈 캠페인 — 얼리버드/CTA 콘텐츠, 성과 분석
